# Customer Segmentation using Power BI

## Project Overview

This project focuses on customer segmentation by analyzing customer demographics and purchasing behavior. The dashboard helps identify different customer groups based on age, gender, annual income, and spending score, enabling businesses to make data-driven marketing decisions.

## Objective

- Analyze customer demographics.
- Segment customers based on purchasing behavior.
- Visualize customer distribution and spending patterns.
- Generate actionable business insights.

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Python (Scikit-learn for K-Means Clustering)
- CSV Dataset

## Dataset

The project uses the **Mall Customer Segmentation Dataset**, which contains:

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

## Dashboard Features

- Total Customers KPI
- Average Annual Income
- Average Spending Score
- Gender Distribution
- Age Group Analysis
- Income Distribution
- Customer Segmentation using K-Means Clustering
- Interactive Filters (Gender, Age Group, Income Group)

## Business Insights

- Identified different customer segments based on spending behavior.
- Compared spending patterns across age groups.
- Analyzed customer distribution by gender.
- Observed the relationship between annual income and spending score.
- Highlighted high-value customer groups for targeted marketing.
